About This Project

What is Phishing?

Phishing is a type of cyber attack where attackers attempt to trick users into revealing sensitive information such as passwords, credit card numbers, or personal data. Phishing attacks often involve malicious websites that mimic legitimate ones, tricking users into entering their credentials.

How Does This Detector Work?

This phishing URL detector uses machine learning to analyze various features of a URL and determine if it's likely to be a phishing site. The detector extracts features such as:

  • URL length and structure
  • Domain age and registration information
  • Presence of suspicious keywords
  • Use of URL shortening services
  • SSL certificate validity
  • Presence of login forms
  • And many more...

The detector also integrates with Google Safe Browsing API to check if a URL has been previously reported as malicious.

How to Stay Safe

Even with tools like this, it's important to practice good cybersecurity habits:

  • Be cautious of unexpected emails or messages asking for personal information
  • Check the URL in your browser's address bar before entering sensitive information
  • Look for HTTPS and a padlock icon in your browser
  • Use password managers to avoid manually entering credentials
  • Keep your software and browsers updated

Technology Stack

This project is built using:

  • Python with Flask for the backend
  • Scikit-learn for machine learning
  • Bootstrap for the frontend
  • Google Safe Browsing API for additional checks
